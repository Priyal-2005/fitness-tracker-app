## 🚀 Setup & Build

### Step 1 — Install dependencies
```bash
cd priyal-fitness
npm install
```

### Step 2 — Run locally (dev)
```bash
npm run dev
```
Opens at: http://localhost:5173

### Step 3 — Build for production
```bash
npm run build
```
Output in: `dist/` folder

### Step 4 — Preview build locally
```bash
npm run preview
```
Opens at: http://localhost:4173

## 📱 Install on Android Phone

### Option A — Android Chrome (Recommended)
1. Run `npm run build && npm run preview`
2. Find your PC's local IP:
   - Windows: `ipconfig` → look for IPv4 Address (e.g. 192.168.1.5)
   - Mac/Linux: `ifconfig` or `ip a`
3. Open Chrome on your phone
4. Go to: `http://192.168.1.5:4173`
5. Tap the **3-dot menu → "Add to Home Screen"** (or "Install App")
6. Done! Open from home screen — works offline.

### Option B — Deploy to Netlify (Free, easiest for permanent install)
1. Run `npm run build`
2. Go to https://app.netlify.com → "Add new site" → "Deploy manually"
3. Drag the `dist/` folder into Netlify
4. Get your URL (e.g. https://priyal-fitness.netlify.app)
5. Open on phone → Install from Chrome

## 🔌 Offline Support

The service worker (auto-generated by vite-plugin-pwa) caches:
- All JS, CSS, HTML assets
- Icons and static files

After first load, the app works **100% offline** — no internet needed.

---

## 💾 Data Storage

All data is stored in **localStorage** on your device:
- `gym_checks_YYYY-MM-DD` — daily exercise checkboxes
- `exercise_weight_log` — weight used per exercise (persists forever)
- `weight_history` — body weight log (up to 90 entries)
- `goal_weight` — your target weight
- `meal_done_YYYY-MM-DD` — daily meal toggles
- `supp_checks_YYYY-MM-DD` — daily supplement tracking
- `daily_checklist` — workout/protein/meals checklist
- `streak_data` — consecutive day streak
- `weekly_history` — per-week completion data

Data resets per-day where appropriate (checklists, meal toggles).
Weight log and strength log persist indefinitely.
